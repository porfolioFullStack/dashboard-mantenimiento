# assets/

Carpeta destinada exclusivamente a **recursos estáticos** del proyecto.

---

## Estructura

- `img/`  
  Imágenes generales del dashboard (placeholders, ilustraciones, etc.)

- `icons/`  
  Íconos utilizados en la interfaz (SVG o PNG)

- `fonts/`  
  Fuentes tipográficas locales (opcional)

---

## Convenciones

- No incluir código (HTML, CSS, JS)
- Usar nombres descriptivos y consistentes
- Evitar espacios y caracteres especiales en nombres de archivo
- Mantener rutas simples y predecibles
